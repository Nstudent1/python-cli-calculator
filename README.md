# Python CLI Calculator

A simple command-line calculator built with Python.  
This project demonstrates basic Python programming concepts such as user input validation, control flow, functions, and error handling.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation for operations
- Error handling for invalid numeric input
- Continuous calculation loop until the user exits

## Technologies Used

- Python 3

## Project Structure

```
python-cli-calculator
│
├── calculator.py
└── README.md
```

## How to Run

1. Clone the repository:

```
git clone https://github.com/Nstudent1/python-cli-calculator.git
```

2. Navigate into the project folder:

```
cd python-cli-calculator
```

3. Run the program:

```
python calculator.py
```

## Example Usage

